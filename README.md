# level2-weather_app.py
📌 Project Description
This project demonstrates how to integrate an external API using Python. The application sends HTTP GET requests to an API, retrieves real-time data, and displays it in a user-friendly format. In this project, APIs can be used to fetch information such as weather details, cryptocurrency prices, or other live data.
🎯 Objectives
Connect a Python application with an external API.
Send GET requests using the requests library.
Retrieve and parse JSON data from API responses.
Display the fetched information in a readable format.
Implement error handling for failed requests and invalid responses.
✨ Features
Real-time data retrieval from an external API.
User-friendly output display.
JSON data parsing and processing.
Error handling and exception management.
Simple and easy-to-understand Python code.
🛠️ Technologies Used
Python
Requests Library
REST APIs
JSON
📚 Learning Outcomes
Understanding API integration in Python.
Working with HTTP requests and responses.
Handling JSON data effectively.
Managing errors and exceptions in API-based applications.
Building real-world applications using external data sources.
